⚖️ LexInsight: Sentimento & Dados

Transformando linguagem jurídica em inteligência estratégica através de NLP e análise de sentimento.

🚀 Visão Geral

O LexInsight é uma aplicação de análise textual orientada a dados, projetada para extrair insights estratégicos a partir de grandes volumes de textos jurídicos.

Utilizando técnicas de Processamento de Linguagem Natural (NLP), o sistema realiza limpeza, estruturação e análise de textos, entregando visualizações claras e métricas objetivas para apoio à tomada de decisão.

🎯 Proposta de Valor
📊 Clareza Analítica: converte textos complexos em visualizações intuitivas
🧠 Inteligência Jurídica: identifica padrões e tendências ocultas
⚡ Eficiência Operacional: reduz drasticamente o tempo de análise manual
📈 Base para Decisão: fornece métricas objetivas para estratégias jurídicas
🧩 Funcionalidades
🔹 Processamento de Texto
Limpeza com Regex
Remoção de Stop Words (incluindo termos jurídicos)
Tokenização
Lematização
🔹 Análise de Sentimento
Classificação de polaridade (positivo, negativo, neutro)
Score numérico (-1 a +1)
🔹 Visualização de Dados
📌 Nuvem de palavras (WordCloud)
📊 Distribuição de sentimentos
📉 Frequência de termos
🔹 Interface Interativa
Upload de textos
Análise em tempo real
Dashboard simples via Streamlit.
